# Employee Management System

ASP.NET Core MVC application to manage employee records with CRUD operations, validation, logging, and SQL Server integration.

* Features
- Employee CRUD operations
- Client-side & server-side validation
- SQL Server with stored procedures
- Bootstrap-based UI
- File logging using Serilog

* Tech Stack
- ASP.NET Core MVC
- SQL Server
- Bootstrap
- jQuery Validation
- Serilog

* Run
1. Open solution in Visual Studio  
2. Update database connection string  
3. Run the project
